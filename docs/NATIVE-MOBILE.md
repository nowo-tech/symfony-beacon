# Mobile apps (Symfony UX Native / Hotwire Native)

> **Removed.** Beacon no longer ships `symfony/ux-turbo` or `symfony/ux-native`.
> In-app Turbo Drive navigation caused unreliable page updates in the operator UI,
> and Hotwire Native depended on Turbo.

Use the **Progressive Web App** (`nowo-tech/pwa-bundle`) for installable mobile access to the same Twig UI. A future native shell can be reconsidered without Turbo Drive if needed.

Historical design notes remain under `specs/008-ux-native/` for reference only.
